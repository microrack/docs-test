# MICRORACK Documentation

This repository contains the documentation content for MICRORACK.org.

## Structure

- `getting-started/` - Introduction and basic setup guides
- `user-guide/` - Comprehensive user documentation
- `tutorials/` - Step-by-step tutorial content

## Content Format

All documentation is written in MDX (Markdown with JSX support) and follows the Fumadocs structure.

Each directory includes a `meta.json` file that defines the page order and titles for navigation.

## Contributing

Documentation updates should maintain the existing structure and follow MDX best practices.
