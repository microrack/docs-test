# Input / Output

The MICRORACK system provides versatile I/O options for connecting to external equipment.

## Audio I/O Modules

### Jack Input Module

The Jack Input module allows you to connect external audio sources:

- **3.5mm input**: For line-level sources
- **6.3mm input**: For instruments and headphone outputs
- **Volume control**: Adjustable input gain
- **Impedance**: High impedance input suitable for guitars and other instruments

### Jack Output Module

The Jack Output module provides audio output to external equipment:

- **3.5mm output**: For connecting to mixers or audio interfaces
- **6.3mm output**: For connecting to amplifiers or headphones
- **Volume control**: Adjustable output level
- **Headphone compatible**: Can drive Specification headphones

## CV I/O

Control voltage signals can be patched externally:

- **CV outputs**: 0-5V range, compatible with most external gear
- **CV inputs**: Accepts Specification modular CV signals
- **Attenuation**: Many modules include input attenuators

## MIDI

The ESP32 Pro Module provides MIDI connectivity:

- **MIDI Input**: Receive note, CC, and clock data
- **MIDI to CV conversion**: Converts MIDI notes to 1V/octave pitch CV
- **CC mapping**: Map MIDI CCs to CV outputs
- **Clock sync**: Sync sequencers to external MIDI clock
