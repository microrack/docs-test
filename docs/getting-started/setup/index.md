# System Setup

Learn how to properly configure your MICRORACK system for optimal performance.

## Power Configuration

1. Check voltage requirements for your region
2. Connect power supply to distribution board
3. Verify all modules receive proper voltage
4. Test system before full operation

## Module Installation

- Mount modules securely in chassis
- Ensure proper spacing for cable access
- Connect power connectors correctly
- Double-check polarity before powering on

## Cable Management

- Use appropriate cable lengths
- Organize cables for easy reconfiguration
- Label complex patches for future reference
- Keep spare cables accessible

## Safety Tips

- Always power off before connecting/disconnecting modules
- Check connections before turning on
- Avoid hot-plugging modules
- Keep system away from moisture
