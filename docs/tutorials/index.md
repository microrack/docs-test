# Tutorials

Follow these step-by-step tutorials to create specific sounds and learn practical techniques.

## Available Tutorials

### [Bass Patch Tutorial](./bass-patch)
Create a powerful, punchy bass sound perfect for electronic music.

**Modules needed**: VCO, VCF, VCA, ADSR
**Difficulty**: Beginner
**Time**: 10 minutes

### [Ambient Pad Tutorial](./ambient-pad)
Design a lush, evolving pad sound for atmospheric music.

**Modules needed**: 2× VCO, VCF, VCA, 2× ADSR, LFO
**Difficulty**: Intermediate
**Time**: 15 minutes

## Tutorial Format

Each tutorial includes:

- Required modules list
- Step-by-step patching instructions
- Parameter settings
- Variation suggestions
- Troubleshooting tips

## Learning Path

1. Start with [Bass Patch](./bass-patch) to learn fundamentals
2. Progress to [Ambient Pad](./ambient-pad) for multi-module patches
3. Experiment with variations
4. Create your own sounds!
