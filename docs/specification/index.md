# Specification

The MICRORACK Standard defines basic technical specifications for any module designers and manufacturers.

> **Source Repository:** [github.com/microrack/specs-test](https://github.com/microrack/specs-test)

## Overview

MICRORACK is designed as an open platform. These specifications ensure interoperability between modules from different designers and manufacturers.

- [Mechanical Specifications](mechanical.md) - Module dimensions, board outlines, power contacts
- [Electronical Specifications](electronical.md) - Voltage rails, signal levels, impedance
