# Chassis

MICRORACK offers several chassis options for organizing and protecting your modules.

## MICRORACK Chassis

The official MICRORACK Chassis provides:

- Integrated power distribution
- Protective enclosure
- Portable design
- Multiple size options

### Available Sizes

| Model | Capacity | Dimensions |
|-------|----------|------------|
| Chassis Mini | 5-8 modules | 15 × 10 cm |
| Chassis Specification | 10-15 modules | 20 × 12 cm |
| Chassis Pro | 20+ modules | 30 × 15 cm |

## Rack Chassis

For integration with Specification rack equipment:

- 19" rack mountable
- 3U height
- Eurorack power compatibility option
- Professional finish

## AE Modular Chassis

Developed in collaboration with Tangible Waves:

- Compatible with AE Modular modules
- Allows mixing MICRORACK and AE modules
- Shared power rail design
- Expandable configuration

## DIY Options

MICRORACK modules work with:

- Specification breadboards (standalone use)
- 3D printed enclosures (community designs)
- Custom wooden cases
- Acrylic laser-cut cases

## Mounting Accessories

| Accessory | Purpose |
|-----------|---------|
| Rubber feet | Anti-slip for desktop use |
| Rack ears | 19" rack mounting |
| Handle kit | Portable carry handle |
| Stacking brackets | Connect multiple chassis |
